
## Usage

```sh

```

### Estou Utilizando
  - eslint

### Criar App React com suporte a TS
```
    create-react-app reactwithts --typescript
```

#### Estrutura

O estado da aplicação é administrado pelo Redux, seguindo a estrutura:

```
  store/
    |->ducks/
         |-> index.ts
         |-> rootReducer.ts
         |-> repositories/
                |-> actions.ts
                |-> index.ts
                |-> types.ts
                |->
```
